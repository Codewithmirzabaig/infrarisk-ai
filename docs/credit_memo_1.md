# Credit Memo 1: Toll Road Infrastructure Project

## Project Overview

This credit memo evaluates a toll road infrastructure project using the InfraRisk AI credit risk framework. The project is assessed using financial, macroeconomic, construction, and sovereign risk indicators.

## Project Type

Toll Road / Transportation Infrastructure

## Key Risk Factors

- Construction delay risk
- Cost overrun exposure
- Traffic demand uncertainty
- Interest rate sensitivity
- Sovereign and regulatory risk

## Financial Metrics

| Metric | Interpretation |
|---|---|
| DSCR | Debt service repayment strength |
| LLCR | Loan life repayment coverage |
| PLCR | Full project life repayment coverage |
| PD | Probability of default |
| LGD | Loss given default |
| EAD | Exposure at default |
| EL | Expected loss |

## Model Assessment

The XGBoost credit risk model evaluates default risk using project-level financial and risk indicators. SHAP explainability identifies the primary risk drivers behind the model prediction.

## Stress Testing

Monte Carlo simulation was used to assess project resilience under 10,000 scenarios involving:

- GDP shock
- Inflation shock
- Interest rate shock
- Traffic demand shock
- Cost overrun shock

## Recommendation

Proceed with caution. The project should maintain adequate DSRA reserves, conservative leverage, and strong EPC protections to mitigate construction and revenue uncertainty.

## Risk Mitigation

- Reduce leverage if DSCR falls below target threshold
- Include tariff adjustment protections
- Maintain contingency reserves
- Monitor sovereign risk indicators
- Use covenant monitoring through InfraRisk AI dashboard
